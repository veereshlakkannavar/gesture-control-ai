# Gesture Control AI

An AI-powered gesture recognition system that allows users to control computer actions using hand gestures.

## Features
- Real-time hand tracking
- Gesture recognition
- Camera-based interaction
- Built using Python and OpenCV

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Future Improvements
- More gesture support
- Volume control
- Presentation control
- Mouse control
